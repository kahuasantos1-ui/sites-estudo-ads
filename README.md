Sites de estudo por matéria do curso de **Análise e Desenvolvimento de Sistemas (ADS)** — Unialfa.

## 📂 Estrutura

```
sites-estudo-ads/
├── index.html              # página índice (landing)
├── cloud-computing/        # ☁️ Cloud Computing
├── design-de-interface/    # 🎨 Design de Interface
└── estrutura-de-dados/     # 🗂️ Estrutura de Dados 1 (Java)
```

Cada matéria é uma pasta com `index.html`, `style.css` e `script.js`.

## 💻 Rodar localmente

É só abrir o `index.html` (da raiz ou de qualquer matéria) com dois cliques. Funciona offline.
