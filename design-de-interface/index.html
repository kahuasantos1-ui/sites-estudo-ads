🤖 Google Stitch cai na Q1 e Q8

Ferramenta do Google que usa IA (Gemini) para gerar interfaces (UI) a partir de prompts de texto ou imagens — e ainda devolve o código.

📌 Finalidade (é só UMA): criação e prototipagem de interfaces em alta velocidade.

Funcionalidades (cite umas 3)

  • Gerar UI por texto — você descreve a tela e a IA cria (o "comando de letrinhas").
  • Atualizar/refinar por comando — ajusta o design em linguagem natural (o "comando de atualização").
  • Exportar — manda o resultado pro Figma ou como código (HTML/CSS).
💡 Pulo do gato: a Q8 pede o "recurso entendido como finalidade principal" — é a mesma resposta da Q1: criação e prototipagem em alta velocidade.
👤 Persona cai na Q2

A persona é um cliente fictício que representa o usuário ideal do serviço, construído a partir de pesquisa real (entrevistas, dados).

Importância para o negócio

  • Define para quem se está projetando — todo mundo do time fala do mesmo usuário.
  • Guia decisões de design e prioriza as funcionalidades que importam pra esse usuário.
  • Evita o erro de "projetar pra si mesmo".
🍔 Exemplo iFood: a persona é o cliente típico (ex.: alguém sem tempo que quer pedir comida rápido). Criar a persona ajuda a decidir quais recursos o app precisa ter.
🔀 User Flow cai na Q3

O User Flow é a representação baseada em cenários de interação: o diagrama do caminho que o usuário percorre entre as telas até atingir o objetivo (ex.: do login até finalizar o pedido).

⚠️ Não confunda: Wireframe é o esboço de uma tela; User Flow é o fluxo que conecta as telas. Na aula vocês montaram no Miro.
🧩 Gestalt — Lei da Proximidade cai na Q4

Gestalt é a teoria da percepção visual: "o todo é mais que a soma das partes". O cérebro agrupa o que vê seguindo princípios.

PrincípioO que diz
Proximidade ⭐Elementos próximos são percebidos como um grupo
SimilaridadeElementos parecidos (cor/forma) são vistos como do mesmo grupo
ContinuidadeO olho segue linhas e curvas contínuas
FechamentoA mente "completa" formas incompletas
Figura-fundoSeparamos o objeto (figura) do fundo
📌 Resposta da prova: o princípio de agrupar por proximidade é a Lei (ou Princípio) da Proximidade.
